🤖 AI Resume Screening System

📌 Description
This project is an AI-based system that automatically screens resumes and matches them with job descriptions to find the best candidates.
🚀 Features
- Upload resumes (PDF)
- Extract text from resumes
- Match resumes with job role
- Rank candidates

- 🛠️ Technologies 
- Python
- Flask
- Machine Learning
- NLP
- Pandas
- Scikit-learn

▶️ How to Run
1. Clone the repository
2. Open in VS Code
3. Install dependencies:
   pip install -r requirements.txt
4. Run the project:
   python app.py
